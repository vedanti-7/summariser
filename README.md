#  DocDigest

DocDigest is a Python‑based document summarization tool that creates concise summaries of long text based on a user‑defined line limit. With a simple web interface and flexible controls, users can quickly generate readable summaries from lengthy documents with ease.

---

##  Features

###  Smart Summarization
- Summarizes documents based on the number of lines specified by the user  
- Extractive summarization logic for key sentence selection  
- Fast processing using Python script logic

###  Web Interface
- Simple UI for easy text input and summary display  
- Built with Flask and HTML templates  
- Static assets for styling and interface elements

###  Flexible Input
- Works with custom text input from users  
- Summaries adjust dynamically based on selected line limit

---

##  Tech Stack

- **Python**  
- **Flask** for web UI  
- Summary logic in `summarize.py`  
- Requirements in `requirements.txt`
